# springboot-project
Project Description

This project is a full-stack web application developed to manage and deliver online learning content in an organized and efficient manner.
The system allows users to access courses, read lessons, attempt quizzes, and interact with AI tools through a modern web interface.

The backend is implemented using Spring Boot, providing RESTful APIs for handling business logic, data processing, and database operations.
The frontend is built using React, ensuring a responsive, dynamic, and user-friendly experience.

The application is designed using layered architecture principles, which improves maintainability, scalability, and code reusability.
